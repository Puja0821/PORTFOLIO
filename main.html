Health Insurance portal

I developed a Health Insurance Portal using React for the frontend and Node.js & Express for the backend. The portal was designed to provide users with an intuitive and seamless experience to manage their health insurance details. It includes features such as viewing insurance policies, claim submissions, and status tracking. For handling employee information and other data, I have integrated a MySQL database, which stores and manages employee records securely and efficiently. The backend uses Node.js with Express to create RESTful APIs that interact with the MySQL database, allowing for smooth data retrieval and updates. This project demonstrates full-stack development skills by combining modern frontend development with React, robust backend API development with Node.js, and reliable relational data management through MySQL.
